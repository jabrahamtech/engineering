## 1. `.vimrc` basics

### Enable line numbers
```
set number
set relativenumber
```

### Indentation
```
set tabstop=4
set shiftwidth=4
set expandtab
set smartindent
```

### Search settings
```
set ignorecase
set smartcase
set incsearch
set hlsearch
```

---

## 2. Leader key

Default leader is `\`, but set it to space:

```
let mapleader = " "
```

Examples:
```
nnoremap <leader>w :w<CR>
nnoremap <leader>q :q<CR>
nnoremap <leader>h :nohlsearch<CR>
```

---

## 3. Useful remaps

Better window movement:
```
nnoremap <C-h> <C-w>h
nnoremap <C-j> <C-w>j
nnoremap <C-k> <C-w>k
nnoremap <C-l> <C-w>l
```

Move lines:
```
nnoremap <A-j> :m .+1<CR>==
nnoremap <A-k> :m .-2<CR>==
```

## 4. Plugin managers

### Recommended:
- **vim-plug**  
- Pathogen  
- Vundle  
- Lazy.nvim (Neovim only)

### vim-plug example:
```
call plug#begin('~/.vim/plugged')
Plug 'tpope/vim-surround'
Plug 'tpope/vim-fugitive'
Plug 'vim-airline/vim-airline'
Plug 'preservim/nerdtree'
call plug#end()
```

Reload and run:
```
:PlugInstall
```
